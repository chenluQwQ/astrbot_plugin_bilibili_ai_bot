## v1.2.0 (2026-06-13)

### ✨ 新增功能

#### 📅 每周总结

- 新增周总结系统：每周固定一天自动回顾这一周的B站生活（看过的视频/番剧、发过的动态、评论区互动），用人设口吻生成总结
- 新增配置项：
  - `ENABLE_WEEKLY_SUMMARY` — 启用每周总结（默认关闭）
  - `WEEKLY_SUMMARY_MODE` — 发送方式（`qq` 私信主人 / `dynamic` 发成B站动态 / `both`）
  - `WEEKLY_SUMMARY_DAY` — 每周生成日（0=周一 ... 6=周日，默认周日）
  - `WEEKLY_SUMMARY_QQ_UMO` — 接收 UMO（留空复用恶意告警的 UMO）
  - `CUSTOM_WEEKLY_INSTRUCTION` — 周总结补充提示词
- 新增 `/bili周总结` 命令手动触发
- 新增 `/biliUMO` 命令：在 QQ 私聊中发送即可获取当前 UMO，并自动写入周总结和恶意告警的 QQ UMO 配置（已有值不覆盖）
- 自动触发安排在睡眠时段、日终清算完成之后，不与清算抢资源

### 🔧 优化

#### 🔗 字幕+热评联合分析

- 修复：开启视频直读/截帧分析时，字幕和热评被完全跳过的问题——现在视觉分析结果会与字幕、热评做最终联合整合（画面与字幕互相印证，冲突时以字幕为准）
- 三条分析路径（视频直读/截帧、封面视觉、纯文字）统一通过 `_build_joint_context` 接入字幕+热评
- 分析提示词加入联合分析指令：以字幕为准概括内容，结尾用1-2句描述评论区氛围/观众反响

#### 📢 推荐主人更日常

- 推荐提示词补全上下文：视频内容概括、看完的感想、推荐理由现在都会传给模型（此前只有标题，导致输出空泛营销腔）
- 明确禁止安利腔/营销腔（"快来看""超好看"、堆感叹号、排比），要求提到视频里具体戳到的点

#### ⚙️ 配置项重排

- 全部 92 个配置项按使用逻辑重新分组排序：账号 → 人设 → 功能开关 → 回复 → 主动行为 → 特别关注 → 番剧 → 动态发布 → 周总结 → 好感度提示词 → 性格演化 → 记忆 → 视觉模型 → 联网搜索 → 图片生成 → 恶意告警 → 系统
- 修正错位项（特别关注不再插在主动行为中间，`VIDEO_SEGMENT_SEC` 归入视频视觉模型组）

## v1.1.31 (2025-05-24)

### ✨ 新增功能

#### 🎬 视频直读多格式兼容

- 新增 `VIDEO_VISION_FORMAT` 配置（下拉选择）
  - `gemini` — Gemini 原生格式（`image_url` + `video/mp4`）
  - `qwen` — 通义千问 / 硅基流动格式（`video_url` + `fps`）
  - `none` — 跳过视频直读，直接走截帧分析（默认）
- 新增 `VIDEO_VISION_FPS` 配置（Qwen 格式专用抽帧率，默认 2）
- 用户自行选择格式，不再自动猜测

#### 🛡️ 恶意告警系统

- 检测到 B站 评论区有人攻击 Bot 时，自动通过 QQ 私信通知主人
- 新增 `ABUSE_ALERT_MODE` 配置（下拉选择）
  - `off` — 关闭（默认）
  - `score` — 根据回复生成时的 `score_delta` 判断
  - `model` — `score_delta` 触发后再调一次模型二次确认，过滤误报
- 新增 `ABUSE_ALERT_QQ_UMO` — 接收通知的 QQ 私聊 UMO
- 新增 `ABUSE_ALERT_SCORE_THRESHOLD` — 触发阈值（默认 -3）
- Bot 会用自己的人设口吻向主人描述被攻击的经过（包含对方 UID），并询问是否拉黑

#### 🔨 新增 LLM 工具：B站拉黑

- 新增 `bili_block_user` 工具
  - 主人在 QQ 端说"拉黑他"等自然语言 → LLM 语义理解 → 调用工具拉黑
  - 参数：`uid`（必填）、`reason`（选填）
  - 自动调用 B站 API + 写入本地黑名单
  - 防护：不能拉黑主人

### 🔧 其他

- 版本号更新至 1.1.31

## v1.1.3 (2025-05-11)

### ✨ 新增功能

#### 📺 番剧观看体系

- 新增 `/bili看番` 命令
  - 支持搜索番剧并选择具体番剧
  - 支持获取番剧详情（评分、简介、更新状态、分集信息等）
  - 支持完整观看单集内容
  - 自动获取并分析：
    - 视频字幕
    - 热评
    - 评论区讨论
    - 用户评价
  - 看完后自动生成番剧观看记忆

- 新增 QQ 工具调用：`霜序去看番`
  - 可在 QQ 聊天中直接要求 Bot 去看番
  - 当前默认只观看 1 集

- 新增 `/bili番剧记忆`
  - 查看番剧观看后的记忆存储结果
  - 用于验证番剧记忆是否正确写入

### 🔎 新增番剧查询工具

- `get_bangumi_timeline`
  - 查询最近新番时间线
  - 支持"最近有什么新番"等自然语言调用

- `get_bangumi_trending`
  - 获取热门番剧排行
  - 支持"番剧排行"等查询

- `search_bilibili(bangumi)`
  - 搜索番剧
  - 支持通过番名模糊搜索

- `get_bangumi_info`
  - 获取番剧详细信息
  - 包括评分、简介、演员、分集、播放状态等

### 🎯 追番系统

- 看完番剧后若评分 ≥ 7 分
  - 自动加入追番列表
  - 日志新增：`📌 追番成功`

- 新增 `get_bangumi_updates`
  - 查询已追番剧更新情况
  - 支持"我追的番更新了吗"等自然语言调用

### 🔧 Bug 修复

- 评论区重复回复修复
  - 新增评论去重逻辑
  - 修复部分情况下 Bot 对同一评论重复回复的问题
