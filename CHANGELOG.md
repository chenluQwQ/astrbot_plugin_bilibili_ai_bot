## v1.1.31 (2025-05-24)

### ✨ 新增功能

#### 🎬 视频直读多格式兼容

- 新增 `VIDEO_VISION_FORMAT` 配置（下拉选择）
  - `gemini` — Gemini 原生格式（`image_url` + `video/mp4`）
  - `qwen` — 通义千问 / 硅基流动格式（`video_url` + `fps`）
  - `none` — 跳过视频直读，直接走截帧分析（默认）
- 新增 `VIDEO_VISION_FPS` 配置（Qwen 格式专用抽帧率，默认 2）
- 用户自行选择格式，不再自动猜测

#### 🛡️ 恶意告警系统

- 检测到 B站 评论区有人攻击 Bot 时，自动通过 QQ 私信通知主人
- 新增 `ABUSE_ALERT_MODE` 配置（下拉选择）
  - `off` — 关闭（默认）
  - `score` — 根据回复生成时的 `score_delta` 判断
  - `model` — `score_delta` 触发后再调一次模型二次确认，过滤误报
- 新增 `ABUSE_ALERT_QQ_UMO` — 接收通知的 QQ 私聊 UMO
- 新增 `ABUSE_ALERT_SCORE_THRESHOLD` — 触发阈值（默认 -3）
- Bot 会用自己的人设口吻向主人描述被攻击的经过（包含对方 UID），并询问是否拉黑

#### 🔨 新增 LLM 工具：B站拉黑

- 新增 `bili_block_user` 工具
  - 主人在 QQ 端说"拉黑他"等自然语言 → LLM 语义理解 → 调用工具拉黑
  - 参数：`uid`（必填）、`reason`（选填）
  - 自动调用 B站 API + 写入本地黑名单
  - 防护：不能拉黑主人

### 🔧 其他

- 版本号更新至 1.1.31

## v1.1.3 (2025-05-11)

### ✨ 新增功能

#### 📺 番剧观看体系

- 新增 `/bili看番` 命令
  - 支持搜索番剧并选择具体番剧
  - 支持获取番剧详情（评分、简介、更新状态、分集信息等）
  - 支持完整观看单集内容
  - 自动获取并分析：
    - 视频字幕
    - 热评
    - 评论区讨论
    - 用户评价
  - 看完后自动生成番剧观看记忆

- 新增 QQ 工具调用：`霜序去看番`
  - 可在 QQ 聊天中直接要求 Bot 去看番
  - 当前默认只观看 1 集

- 新增 `/bili番剧记忆`
  - 查看番剧观看后的记忆存储结果
  - 用于验证番剧记忆是否正确写入

### 🔎 新增番剧查询工具

- `get_bangumi_timeline`
  - 查询最近新番时间线
  - 支持"最近有什么新番"等自然语言调用

- `get_bangumi_trending`
  - 获取热门番剧排行
  - 支持"番剧排行"等查询

- `search_bilibili(bangumi)`
  - 搜索番剧
  - 支持通过番名模糊搜索

- `get_bangumi_info`
  - 获取番剧详细信息
  - 包括评分、简介、演员、分集、播放状态等

### 🎯 追番系统

- 看完番剧后若评分 ≥ 7 分
  - 自动加入追番列表
  - 日志新增：`📌 追番成功`

- 新增 `get_bangumi_updates`
  - 查询已追番剧更新情况
  - 支持"我追的番更新了吗"等自然语言调用

### 🔧 Bug 修复

- 评论区重复回复修复
  - 新增评论去重逻辑
  - 修复部分情况下 Bot 对同一评论重复回复的问题
